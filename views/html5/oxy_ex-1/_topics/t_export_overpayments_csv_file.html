Export overpayments to a CSV file

Overpayments: Export overpayments to a CSV file.

You can export overpayments to a CSV file that you can then open using a spreadsheet program.

  1. Use the filter criteria to display the overpayments you want to export.
  2. Click Actions, and then click Export Overpayments.
  3. Read the message in the Export Overpayment Summary window, and then click Close.

Download an overpayments export file

Overpayments: Download a file containing exported overpayments.

Once you have exported overpayments to a file, you can download the file via a notification.

  1. At the top of the Availity Portal, click Notifications, and then locate the notification with the following title:

    The Overpayments Report you ran on mm/dd/yyyy at hh:mm am/pm is ready to download.

    where mm/dd/yyyy and hh:mm am/pm are the date and time the overpayments were exported.

    Note: If you do not see the notification, click View All, and then locate the notifiation on the Notifications Center page.
  2. Click the notification, and then follow your browser's prompts to open or save the CSV file.
