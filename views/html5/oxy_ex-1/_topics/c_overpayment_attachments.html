Overpayment file attachments

Overpayments: Manage file attachments for overpayments.

You can attach files containing supporting documentation to overpayments. You can also download and delete files that are attached to overpayments. You can only attach files to overpayments that have a status of Inquired, Disputed, or Resolved.

You can attach files in the following image formats:

  • PDF Adobe portable document format

  • GIF graphics interchange format

  • JPG, JPEG joint photographic experts group

  • PNG portable network graphics

  • TIFF, TIF tagged image file format

Attach a file to an overpayment

Overpayments: Attach a file to an overpayment.

To attach a file to an overpayment:

  1. Click an overpayment card.
  2. Click Add an attachment (if the overpayment has no attachments) or Add another attachment (if the overpayment as one or more attachments).
  3. In the file upload dialog box, select the file to attach, and then click Open.
When you attach a file to an overpayment, a paper clip icon displays in the overpayment card.

Download an overpayment file attachment

Overpayments: Download a file attached to an overpayment.

To download a file attached to an overpayment:

  1. Click the overpayment card that has the file attachment you want to download.
  2. Click Download next to the file attachment.
  3. Follow your browser's prompts to save the file attachment.

Delete a file attachment from an overpayment

Overpayments: Delete a file attachment from an overpayment.

To delete a file attachment from an overpayment:

  1. Click the overpayment card that has the file attachment you want to delete.
  2. Click Delete next to the file attachment.
