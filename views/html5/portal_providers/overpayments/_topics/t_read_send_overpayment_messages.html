Read and send messages about overpayments

Overpayments: Communicate with the payer about overpayments.

You can send messages about an overpayment to the payer and read messages about an overpayment from the payer. Messages are saved to the overpayment's conversation history, which includes the following information:

  • All messages about an overpayment that you send to the payer.

  • All replies about an overpayment that you receive from the payer.

  • The question or comments you entered when you requested information about a new overpayment.

  • The reason or description you entered when you disputed an overpayment.

  • To view the messages in an overpayments conversation history, do one of the following:

    • Click the overpayment card.

    • Click the action menu in the overpayment card, and then click View Conversation.
    Tip: You can locate overpayments that have unread messages using the Search By fields near the top of the Overpayments page. Select Recent Updates in the first search field, and then select Unread Messages in the second search field
  • To send a message about the overpayment to the payer, type the message in the Send Message field, and then click Send Message.
