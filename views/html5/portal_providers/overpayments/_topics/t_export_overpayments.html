Export overpayments

Overpayments: Export overpayments to a CSV file or a PDF file.

You can export overpayments to to the following file formats:

  • A CSV file that you can then open using a spreadsheet program.

  • A PDF file that you can open using Adobe Acrobat Reader.

Export overpayments to a PDF file

Overpayments: Export overpayments to a PDF file.

You can export overpayments to a PDF file that you can then open using Adobe Acrobat Reader.

  1. Click the action menu in the overpayment card, and then click Download PDF.
  2. Follow your browser's prompts to open or save the PDF file.
