Overpayments

You can use the overpayments application to view overpayments and perform the following actions on overpayments:

  • Request more information about an overpayment.

  • Dispute an overpayment.

  • Resolve an overpayment.

  • Attach supporting documentation and other files to an overpayment.

  • Send messages about an overpayment to the payer.

  • Reply to messages sent from a provider.

Access the overpayments app

Click Payments > Overpayments in the navigation bar.

Navigate the overpayments app

Overpayments status bars, badges, and icons

The color of the status bar to the left of an overpayment indicates the overpayment's status.

Status Bar Color Status Description

Black

Action Required

The overpayment is new, and no action has been taken on the overpayment.

Blue

Inquired

The provider has requested more information about the overpayment from the payer.

Yellow

Disputed

The provider has disputed the overpayment with the payer.

Green

Resolved

The provider has resolved the overpayment with the payer.

Gray

Closed

The payer has closed the overpayment. Note: At this time, overpayments are closed by the payer. In a future release, a new feature will enable providers to close overpayments.

The overpayment app displays various icons and badges to help you easily identify certain information.

  • A new updates badge displays when an overpayment has been updated with new messages or file attachments.

  • The message icon displays on overpayments that have messages.

  • A badge next to the message icon identifies the number of new messages for an overpayment.

  • The attachment icon displays on overpayments that have attachments.

  • Click the action menu next to an overpayment to display a menu of different actions you can perform, such as resolve an overpayment or dispute an overpayment.

  • Click an overpayment card to display more information about the overpayment, send a message about the overpayment to the payer, or add an attachment to the overpayment.

Search and sort overpayments

Overpayments: Search for and sort overpayments.

Use the fields near the top of the Overpayments page to search for overpayments and sort the overpayments displayed on the page.

  • Under Search By, select a filter option in the first field, and then enter or select filter criteria in the second field. For some search options, you must click Search to apply the filter.
    Search option:Search for:
    Assigned To

    Overpayments assigned to you.
    Claim Number

    Overpayments associated with a specific claim number.
    Invoice

    Overpayments associated with a specific invoice number.
    Overpayment Type

    Contractual overpayments or non-contractual overpayments.
    Patient Last Name

    Overpayments associated with a patient's last name.
    Payer

    Overpayments associated with a specific payer.
    Recent Updates Overpayments that have unread messages.
    Status

    Overpayments with a specific status.

    Note: Select All Active to display overpayments with these statuses: Action Required, Inquired, Disputed. Select All Inactive to display overpayments with these statuses: Resolved, Closed.
  • You can combine multiple filters. For example, you can filter by claim number and by one or more overpayment statuses. Each combination of filter option and filter criteria that you apply displays next to the Filters label.
  • To remove a filter, click the X next to the filter.
  • In the Organization field, select the organization whose overpayments you want to display.
  • In the Sort By field, select an option to sort overpayments by oldest or newest.
