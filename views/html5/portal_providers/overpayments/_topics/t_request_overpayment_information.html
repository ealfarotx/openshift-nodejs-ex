Request information about a new overpayment

Overpayments: Request informaton about an overpayment from the payer.

You can request information about an overpayment from the payer. You can only request information about overpayments that have an Action Required status.

  1. Click the action menu next to the overpayment, and then click Request more information.
  2. In the Overpayment Inquiry window, type the question or comment you want to send to the payer. The question or comment you enter will be saved as part of the overpayment's conversation history.
  3. Click Submit.
  4. In the Success window, click Close, or click Inquired Overpayments to jump to a list of inquired overpayments.

When you request information about an overpayment, the overpayment's status is changed to Inquired.
